# Agentic-CliniAI
> 致力于创建安全、透明、可溯源的AI诊疗工具，辅助医疗专业人员提升诊疗效率与准确性。

[![Website](https://img.shields.io/badge/website-agentic--cliniai.github.io-blue)](https://agentic-cliniai.github.io)

- **Clinical Agentic Engine** - 主诊智能体代理系统
- **Evidence Tracking Agentic Engine** - 循证医学决策支持代理系统
- **SafeGuard Toolkit**- 医疗AI安全框架：幻觉检测、不确定性量化、偏差校验与反馈机制

## 🧩 核心项目

- 🩺 **DermAgent** — 皮肤科多Agent诊断系统 [`dermagent`](https://agentic-cliniai.github.io/dermagent.html)
- 🔁 **Möbius** — 双向互监督框架：充分发挥 SAM 的视觉理解能力与 LVLM 的语义推理能力。两者双向互补，构建可信赖的医学视觉语言系统 [`mobius`](https://github.com/agentic-cliniai/mobius)
- 🛡️ **SafeGuard Toolkit** — 医疗AI安全框架 [`safeguard`](https://github.com/Agentic-CliniAI/Safeguard-Toolkit)
