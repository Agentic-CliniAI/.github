## 🎯 使命

Agentic-CliniAI 致力于创建安全、透明、可溯源的AI诊疗工具，辅助医疗专业人员提升诊疗效率与准确性。

## 📦 核心项目

- **Clinical NLP Engine** - 临床文本自然语言处理引擎
- **MedDecision Agent** - 循证医学决策支持代理
- **SafeGuard Toolkit** - 医疗AI安全框架
